    Project Description
    Using a framework like bootstrap that styles a site, we will build a website that will help first year Villanova students (or undecided students) to discover their potential majors and find information about that major easily. This website will require them to add information about themselves and the site will generate helpful links that direct them to information about the major, such as: required courses, professors, etc. The site will also allow them to take a quiz if they are undecided about their major. We know that navigating through the Villanova web pages might be difficult and confusing for first years so hopefully with this website, it will make it easier for them to navigate and find what they need to know.
